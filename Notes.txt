TROUBLESHOOTING:
- If GD not loaded: Upload still works (saves original size, no crop/resize)
- Enable GD in php.ini by uncommenting: extension=gd
- Check admin/photo_upload_test.php for GD library status
- Upload directory must be writable (check /assets/images/employees/ permissions)
- Max upload size configured to 5MB (adjust in EmployeePhoto class if needed)
